## 디지털영상처리및실습 이미지 전처리 단계(예정)

### pdf <-> img 변환 (필요 시)
* PDF → 이미지: pdf2image 모듈 사용
* 이미지 → PDF: Pillow (PIL.Image) 사용
### 이미지 그림자 제거 (필요 시)
* 1. 문서 평탄화(Background Subtraction) + CLAHE: 흑백으로 나옴, OCR에 가장 적합
* 2. 조명 그림자 분리(Morphology 기반): 컬러 이미지 그대로 남는 방식. 그림자 제거만 원하는 사용자에게 적합함
* 3. Gray World(조명 보정 + 가우시간 블러): 그림자 부분이 어둡고 주변이 밝은 차이를 이용하여 밝기를 평균화하는 방식, 흑백 이미지 처리에 좋음  
* 4. Adaptive Threshold: 그림자 무시 후 바로 이진화 하므로 글자 aliasing이 심해짐(사람이 읽기도 힘들정도로 깨짐)
